# Aviation Weather Board
Aviation Weather Board

### How to install (Raspberry Pi)

`curl -s https://raw.githubusercontent.com/eelcohn/AWB/main/rpi/install.sh | sudo bash`

### Keyboard shortcuts

| Key | Description |
|-----|-------------|
|  I  | Show IP address |
|  R  | Force a refresh of weather data |
